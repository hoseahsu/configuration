# hexo

## 配置环境

#### node

[Node.js](https://nodejs.org/en/)

#### git

```bash
sudo apt install git
```

## 安装

```
npm install hexo-cli -g
hexo init blog
```
## 部署

```
cd blog
npm install
hexo server
```
